# Python Examples

An interactive, Python Learning course for beginners all on our browser. No downloads, no paywalls. Its broken into small units with the basic fundamental skills, with tons of different lessons including code examples, lesson goals, mini quizzes, practice questions, and projects. Basically everything you need to finally learn Python!!!

## How to Try It

Its super simple. just open any lesson file into your browser, you dont need to install anything, no server needed. all of your data stays locally so no data risks,

```
Unit 1/lesson 1/lesson 1.html
```

## How to Quick Start

1. Clone my repository or download it from my Github [https://github.com/DagaVedant/Python-Examples](https://github.com/DagaVedant/Python-Examples).

2. Open any `Lesson #.html` file in your browser.

3. Read the lesson information, use the code example, then take the lesson quiz to check your understanding, and then try the practice problems.

You don't even need Python to follow the along inside the browser, but if you want to run the practice code, you need to have Python 3.x installed locally on any IDE of your choice, I love VS Code, but any other IDE is also perfect.

## How does it Work

Basically, each lesson has its own HTML file that automatically loads all the contenct from a `data.js` file. The JS file exports a `LESSON` object that has all the information for the unit like title, code, quiz, practice problems, and make a projec. The HTML renders all the of the content automatically.
